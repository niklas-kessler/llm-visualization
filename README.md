## Getting Started

1. Clone and download this repository.
2. Create a system environment variable called OPENAI_API_KEY and insert your OpenAi API key as the value. Do NOT replace the line of code directly so that it is not accidentally published.
3. Run npm install i to install all required packages.
4. Run the development server: npm run dev.
5. The application will run on [http://localhost:3000](http://localhost:3000).
