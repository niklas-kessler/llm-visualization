In the Prompts-folder, we document used prompts.

Moreover, the reasoning trees generated in the experiments have been saved as json, and can 
be found in the Results-folder. They can be investigated manually or loaded into GPT-Viz. For 
the latter, copy paste the json object into the input field that pops up when you click the 
H-button in the window selection bar at the top of the window and click on send.

GPT-Viz can be cloned from https://github.com/niklas-kessler/llm-visualization.
Note that the most current version is on the branch "final-fixes(BA)".
